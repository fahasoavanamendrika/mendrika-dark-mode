🌙 Qu'est-ce que le Dark Mode ?

Le dark mode (ou mode sombre) est une version visuelle d’un site web qui utilise des couleurs foncées, souvent un fond noir ou gris foncé avec du texte clair. Il améliore le confort visuel, surtout dans les environnements sombres, et peut économiser de la batterie.

🧱 1. La structure de la page

Une page web est construite en HTML, qui définit les éléments comme les titres, les paragraphes ou les boutons. Pour activer un mode sombre, on ajoute généralement un bouton que l'utilisateur peut cliquer pour changer l’apparence du site.

🎨 2. Le style visuel

Les couleurs de la page sont gérées par le CSS. En mode clair, on utilise des couleurs claires pour le fond et foncées pour le texte. En mode sombre, c’est l’inverse. Le site applique un style différent en changeant une classe sur la page.

⚙️ 3. Le rôle du JavaScript

Le JavaScript permet de rendre la page interactive. Il détecte le clic sur le bouton et change l’apparence de la page en ajoutant ou enlevant une classe spéciale.

💾 4. Garder le mode choisi

Pour mémoriser la préférence de l’utilisateur (sombre ou clair), on peut utiliser une fonction appelée localStorage. Cela permet de conserver le mode choisi même après avoir fermé ou rechargé la page.

✅ En résumé

Le dark mode fonctionne grâce à 3 éléments : HTML pour la structure, CSS pour le style, et JavaScript pour l’interactivité. C’est une solution moderne et pratique, de plus en plus utilisée sur le web.
